The minimum value of moving average is equal to 0.35 whereas the maximum value reached 5.44

Recommendations:

  • Set a new parameter which will monitor a size change of a portfolio in time (dynamically) and when a certain threshold is reached - update the parameter accordingly for upcoming month.
  • Create a model based on historical data which will predict a change in a following month with a certain risk level.
